# fxdx Contracts
Contracts for fxdx.

Docs at https://fxdx.gitbook.io/a/evm/contracts.

## Install Dependencies
If npx is not installed yet:
`npm install -g npx`

Install packages:
`npm i`

## Compile Contracts
`npx hardhat compile`

## Run Tests
`npx hardhat test`
